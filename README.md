# Say Less · 少废话

A lightweight Chrome extension that optimizes prompts to reduce token usage.

## ✨ Features

* Generate 3 versions of your prompt:

  * Chinese (CN)
  * English (EN)
  * Mixed (MIX)
* Compare token usage
* Highlight the most efficient version

## 🚀 How to use

1. Clone this repo
2. Open Chrome → `chrome://extensions/`
3. Enable "Developer mode"
4. Click "Load unpacked"
5. Select this folder

## 🔑 Setup

Replace the API key in `popup.js`:

```
const API_KEY = "YOUR_API_KEY_HERE";
```

## 💡 Concept

Different languages use tokens differently.
This tool helps you find the most efficient way to express your prompt.

## 🧠 Idea

Say less. Spend less.
